Motor Vechicle Collision In Newyork

This was a project I started back in October 2019 when I first learned python and its frame work .

In this project I tried to analyse the newyork vechicle collision data. In this Dashboard we can get that where the most people injured in NYC.

We can also get that how may collision occur during a given time of day. and we can even see th Raw data in 1 hour of spot.

By this all we can finally conclude that which street are dangerous so we can follow safety on that street at that time.

This Project is one of the project from Coursera guided project.
